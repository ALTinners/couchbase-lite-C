◆ cbl_blob_createWithStream()

CBLBlob* cbl_blob_createWithStream ( const char *  contentType,
CBLBlobWriteStream *  writer 
)

Creates a new blob after its data has been written to a CBLBlobWriteStream.

You should then add the blob to a mutable document as a property – see CBLMutableDict_SetBlob and CBLMutableArray_SetBlob.

Note
You are responsible for releasing the CBLBlob reference.
Do not free the stream; the blob will do that.
Parameters
contentTypeThe MIME type (optional).
writerThe blob-writing stream the data was written to.
Returns
A new CBLBlob instance.

CBLBlobWriteStream* cbl_blobwriter_new ( CBLDatabase *  db,
CBLError *  outError 
)

Opens a stream for writing a new blob.

You should call cbl_blobwriter_write one or more times to write the data, then cbl_blob_createWithStream to create the blob.

If for some reason you need to abort, just call cbl_blobwriter_close.
