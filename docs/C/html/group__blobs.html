CBLBlobWriteStream* cbl_blobwriter_new ( CBLDatabase *  ,
CBLError *  outError 
)

Opens a stream for writing a new blob.

You should call cbl_blobwriter_write one or more times to write the data, then cbl_db_createBlobWithStream to create the blob, then cbl_blobwriter_free.

If for some reason you need to abort, just free the writer without calling cbl_db_createBlobWithStream.
